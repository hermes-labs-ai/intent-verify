# Code of Conduct

This project follows the Contributor Covenant style standard:

- be respectful
- assume good faith
- focus on concrete technical discussion
- avoid harassment, insults, or personal attacks

Unacceptable behavior includes:

- abusive language
- discriminatory behavior
- intimidation
- doxxing
- bad-faith disruption

Maintainer actions may include warning, moderation, or removal from the project space.
